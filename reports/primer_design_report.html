Methodology

Primer Design

PCR primers were designed to amplify regions containing variant(s) in the following gene(s) and transcript(s): NBN (NM_002485.4), CHEK2 (NM_007194.4), BRCA1 (NM_007294.4), APC (NM_000038.6), ATM (NM_000051.4). A total of 5 variant(s) were analysed: NBN NM_002485.4:c.657_661delACAAA; CHEK2 NM_007194.4:c.470T>C; BRCA1 NM_007294.4:c.116G>A; APC NM_000038.6:c.7519C>T; ATM NM_000051.4:c.3848T>C.

All genomic coordinates were referenced to the GRCh38/hg38 human genome assembly. Coding DNA sequence (CDS) positions were mapped to genomic coordinates using the Ensembl REST API (https://rest.ensembl.org) and cross-validated against NCBI Entrez (https://eutils.ncbi.nlm.nih.gov/entrez/). Transcript annotations were verified against the MANE (Matched Annotation from NCBI and EBI) database to ensure use of clinically recommended reference sequences.

Genomic sequences were retrieved from the Ensembl REST API, extending 250 bp on each side of each variant position to provide a search space for primer placement.

Population Variant Masking

Known population polymorphisms were obtained from the Ensembl Variation API (sourced from gnomAD v4). Positions with a minor allele frequency (MAF) ≥ 0.500% in the global (worldwide) population were masked with IUPAC ambiguity code "N" in the template sequence provided to the primer design engine. This masking ensures that primers do not bind at polymorphic positions, which could otherwise lead to allele-specific amplification bias or failure.

Primer Design Engine

Primers were designed using the Primer3 engine (via the primer3-py Python interface) with the following key parameters: amplicon size 100–500 bp, primer melting temperature (Tm) 57.0–64.0 °C (optimal 60.0 °C), primer length 18–27 nt (optimal 22 nt), GC content 40.0–60.0% (optimal 50.0%). Thermodynamic calculations used the SantaLucia nearest-neighbour model with salt corrections for 50.0 mM monovalent cations, 1.5 mM Mg2+, 0.6 mM dNTPs, and 50.0 nM template DNA. Maximum self-complementarity was set to 8.0, 3' end complementarity to 3.0, and pair complementarity to 8.0 (Primer3 thermodynamic alignment scores).

Primer placement was constrained to enforce a minimum distance of 40 bp between the nearest primer edge and the variant position.

Pseudogene Discrimination

To minimise the risk of pseudogene co-amplification, a BLAST+-based homology analysis was performed for each target region. Flanking sequences were queried against the human genome using BLASTn (word size 11, evalue ≤ 1e-10, minimum identity ≥ 85.0%, minimum aligned length ≥ 50 nt) to identify homologous regions (pseudogenes, segmental duplications). No significant homologous regions were detected for any of the target loci, so standard Primer3 ranking was used.

Specificity Verification

Primer specificity was verified using: UCSC In-Silico PCR (https://genome.ucsc.edu/cgi-bin/hgPcr), which simulates PCR amplification against the full genome assembly (GRCh38/hg38) to detect off-target binding sites. Continuous design mode was used: for each variant, up to 20 candidate primer pairs were generated by Primer3 and evaluated sequentially for specificity until at least 3 verified-specific pair(s) were found per variant, or all candidates were exhausted.
